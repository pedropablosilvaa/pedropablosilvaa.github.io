Data Analysis with DuckDB

In this project, I explored the capabilities of DuckDB for processing analytical queries on local files.

Goal

To reduce the time it takes to aggregate millions of rows of CSV data.

Results

DuckDB proved to be orders of magnitude faster than traditional Pandas implementation for this specific use case.

Code Snippet

import duckdb
con = duckdb.connect()
con.execute("SELECT avg(a) FROM 'data.csv'")
print(con.fetchall())

React Personal Portfolio

This project demonstrates how to combine the static site generation power of Hugo with the interactivity of React.

Tech Stack

  • Hugo: For content management and build speed.
  • React: For the landing page and interactive components.
  • Vite: For bundling the React application.

Challenges

Integrating React into a Hugo pipeline required setting up a proper build step where Vite outputs the assets to Hugo’s static folder.

Outcome

A fast, SEO-friendly website with a modern developer experience.
